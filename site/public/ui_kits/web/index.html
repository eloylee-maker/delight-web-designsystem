delight.ai · web UI kit

Components used across the marketing site.

Recreated from the production codebase. All components use Delight Warm tokens, Helvetica Now Text body, Serrif heads, and a single black/lime accent system.

Global navigation

Fixed top bar, 72px tall, white background. Hover reveals dropdown panels (Capabilities mega-menu, Solutions/Resources floating, language popover). Customers has no dropdown — hover only paints a 2px black underline.

Buttons

Three hierarchies — Primary (black), Secondary (outlined), Tertiary (underlined link). Hover slides the label 10px left and fades a → arrow in from the right. Order in a row is always Secondary → Primary.

Footer

Landing patterns

Hero, feature stack, stat band, dark CTA banner, and FAQ — the building blocks of every marketing page. See the full reference at preview/landing-patterns.html.

Page rhythm
  1. Hero — bg-white. Eyebrow, serif H1 (88/96px), one-line lede, Secondary→Primary CTA pair.
  2. Feature section — bg-subtle. 2-up or 3-up cards on white tiles inside a tinted band.
  3. Stat band — bg-white. Bare numbers (Serrif, 88px), short labels underneath.
  4. CTA banner — bg-black. Serif H2 in white, lime button (the only place lime acts as a CTA).
  5. FAQ — bg-white. Hairline-divided rows, plus icon rotates to ✕ on open.
