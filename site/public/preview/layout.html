Header — single global section, top of page

Communicates the page's identity, key message, and primary CTA at first glance. A page must not contain multiple headers.

Body — supporting region

Expands the Header through detailed information, structured explanations, and examples. Body text stays 2–3 lines and never exceeds a 6-column width. Must not repeat the core message of the Header.
